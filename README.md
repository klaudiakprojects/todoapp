# Todo Application written in JavaScript with E2E testing in Cypress

This is a simple to-do app built with HTML, CSS, and JavaScript. It allows you to create, read, update, and delete tasks. The app stores tasks in the browser's local storage, so your tasks will still be there even if you close the browser window.

The app is hosted at https://klaudiakprojects.github.io/todoapp, so you can check it out there.

## Getting started
To use the app go to https://klaudiakprojects.github.io/todoapp in your web browser.

## Using the app
To add a new task, simply enter the task description in the input field and click the "Add" button. The task will be added to the list.

To update a task, click on the task you want to update. The task description will be replaced with an input field where you can edit the description. Press Enter to save your changes.

To delete a task, click the "Delete" button next to the task you want to delete. The task will be removed from the list.

The app uses the browser's local storage to store the tasks, so your tasks will still be there even if you close the browser window.

## Testing
The app includes tests using Cypress.
